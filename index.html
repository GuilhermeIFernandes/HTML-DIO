O que é HTML?

HTML é a linguagem usada para criar e estruturar páginas web. Utiliza elementos representados por tags, para formatar o texto, criar links, inserir imagens, e outras funcionalidades.

Como funciona?

É um conjunto de elementos que você usa para marcar e organizar diferentes partes do conteúdo para que elas apareçam ou funcionem de uma maneira específica. Cada elemento HTML é representado por uma tag. As tags geralmente vêm em pares, uma de abertura e uma de fechamento , e elas marcam o início e o fim do elemento.

Estrutura básica

  1. <!DOCTYPE html>: Indica que o documento está usando HTML5.
  2. <html lang="pt-BR">: Envolve todo o conteúdo da página e define o idioma como português brasileiro.
  3. <head>: Contém metadados (informações sobre o documento).
    • <meta charset="UTF-8">: Define a codificação de caracteres.
    • <meta name="viewport" content="width=device-width, initial-scale=1.0">: Ajusta a visualização para dispositivos móveis.
    • <title>: Define o título da página que aparece na aba do navegador.
  4. <body>: Contém todo o conteúdo visível da página.

Tags

Texto

  • <h1> até <h6>: Usadas para definir cabeçalhos. <h1> é o mais importante e maior, enquanto <h6> é o menor e menos importante.
  • <p>: Define um parágrafo de texto.
  • <mark>: Destaca o texto como se estivesse marcado com um marcador.
  • <small>: Indica texto pequeno.
  • <i>: Define texto em itálico.
  • <u>: Define texto sublinhado.
  • <strong>: Define texto importante, geralmente exibido em negrito.
  • <sub>: Define texto subscrito.
  • <sup>: Define texto sobrescrito.
  • <blockquote>: Define uma seção que foi citada de outra fonte.
  • <abbr>: Define abreviações ou acrônimos. Nessa tag é utilizado o atributo "title" que indica o nome completo da abreviação que será exibido quando o mouse está em cima do texto.
  • <hr>: Insere uma linha horizontal, geralmente usada para separar conteúdo. Ela é uma tag autoencerrada, ou seja, não tem tag para fechamento.

Lista

  • <ol>: Define uma lista ordenada.
  • <ul>: Define uma lista não ordenada.
  • <li>: Define um item de lista.

Em listas ordenadas (<ol>) é possível utilizar o atributo "type" que altera como será exibido os números que ordenam a lista.

OBS.: Só é possível alterar o ícone de listas não ordenadas (<ul>) por meio de CSS.

Link

  • <a>: Define um hyperlink, usado para vincular a outra página ou recurso.

A tag <a> tem os seguintes atributos:

  • "href" (obrigatório): Define o destino do link. Pode ser uma URL ou um caminho de arquivo.
  • "target": Define onde o link será aberto. Os mais utilizados são:
    • "_self" (padrão): Abre o link na mesma aba/janela.
    • "_blank": Abre o link em uma nova aba/janela.
  • "title": Informa um texto que aparece quando o usuário passa o mouse sobre o link.

Imagem

  • <img>: Adiciona uma imagem à página HTML. Ela é uma tag autoencerrada. Os principais atributos dessa tag são:
    • "src" (obrigatório): Especifica o caminho da imagem. Pode ser uma URL ou um caminho de arquivo.
    • "alt" (obrigatório pela acessibilidade): Informa um texto alternativo que descreve a imagem. Esse texto é exibido se a imagem não puder ser carregada e é utilizado para leitores de tela, ajudando na acessibilidade.
    • "title": Informa um texto que aparece quando o usuário passa o mouse sobre o link.
    • "width" e "height": Definem a largura e a altura da imagem em pixels.
